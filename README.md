# 🛒 E-Commerce Mobile Application

---

## 📖 Project Overview

The E-Commerce Mobile Application is a cross-platform shopping application developed using **React Native** and **Expo**. The application allows users to securely browse products, manage their shopping cart, maintain a wishlist, complete purchases, and view previous orders through an intuitive and responsive mobile interface.

The application integrates **Firebase Authentication** for secure user login and registration, while **Firebase Firestore** is used for cloud-based storage of products, user information, shopping carts, wishlists, and order history.

Designed using a component-based architecture, the application demonstrates modern mobile development practices including responsive UI design, cloud database integration, secure authentication, and state management.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/JashUnalkat/ecommerce-mobile-app.git
```

Navigate to the project folder

```bash
cd ecommerce-mobile-app
```

Install dependencies

```bash
npm install
```

Run the application

```bash
npx expo start
```

Open the application using:

- Expo Go
- Android Studio Emulator
- Physical Android Device
